# JQuery File Chooser
Plugin JQuery to show filechooser.

## How to use
```
  <input type="file" class="file-chooser" multiple>
  
  <input type="file" class="file-chooser">
  
  <script>
    $('.file-chooser').simpleFileChooser();
  </script>
```
